# Drive a Car Simulator — Concept Art & Visual Style Guide

## Overview

This folder contains the full concept art package, visual style guide, VFX implementation guide, and sound design direction for Drive a Car Simulator. Assets cover all four worlds (Grasslands, Scorching Desert, Frozen Tundra, Neon City) and define the complete audio-visual design language for 1.0.

## Art Direction Summary

- **Style:** Low-poly, vibrant, hyper-saturated cartoon
- **Mood:** Bright, energetic, "first day of summer"
- **Reference:** Pet Simulator X meets arcade racing
- **Target:** Young Roblox players — every screen must grab attention in thumbnails
- **Worlds:** Grasslands → Scorching Desert → Frozen Tundra → Neon City (each with unique palette)
