## 3. Rules Template

Post in `#rules`:

```
SERVER RULES

Please follow these rules to keep our community welcoming for everyone.

1. Be Respectful
   No harassment, hate speech, discrimination, or personal attacks. Treat everyone the way you'd want to be treated.

2. No Spam
   Don't flood channels with repeated messages, excessive caps, or meaningless content. This includes emoji spam and copypasta.

3. No Self-Promotion
   Don't advertise other Discord servers, games, or social media without permission. Content creators: use #content-creators.

4. Keep It Family-Friendly
   This is a Roblox community. No NSFW content, excessive profanity, or inappropriate topics. Roblox's community standards apply here too.

5. No Exploiting or Cheating Discussion
   Don't share, discuss, or promote exploits, hacks, auto-clickers, or any cheating methods. Immediate ban.

6. Use Channels Correctly
   Post in the right channel. Bug reports go in #bug-reports. Screenshots go in #screenshots. Trades go in #pet-trading.

7. No Begging
   Don't beg for Robux, game passes, codes, or free items from staff or other members.

8. Listen to Moderators
   Moderator decisions are final. If you disagree, DM a moderator respectfully — don't argue in public channels.

9. No Leaking or Datamining
   Don't share unreleased content, leaked code, or datamined information. Spoiling upcoming features ruins the surprise for everyone.

10. One Account Per Person
    Alt accounts used to evade bans or manipulate the server will be permanently banned.

CONSEQUENCES:
- 1st offense: Warning
- 2nd offense: 1-hour mute
- 3rd offense: 24-hour mute
- 4th offense: Permanent ban

Severe violations (slurs, NSFW, doxxing, exploit sharing) result in an immediate permanent ban.
```

## 8. Moderator Guidelines

### Role & Responsibilities

Moderators are the first line of community management. Your goals:
1. Keep the server welcoming and on-topic
2. Respond to player issues quickly
3. Escalate bugs and critical feedback to the dev team
4. Enforce rules consistently and fairly

### Moderation Protocol

**Warning System:**
- 1st offense → DM warning + note in `#mod-log`
- 2nd offense → 1-hour mute + `#mod-log` entry
- 3rd offense → 24-hour mute + `#mod-log` entry
- 4th offense → Permanent ban + `#mod-log` entry

**Immediate Ban Offenses (no warnings):**
- Slurs, hate speech, or discrimination
- NSFW content of any kind
- Doxxing or sharing personal information
- Sharing exploits, hacks, or cheat tools
- Scam attempts (fake code sites, phishing links)
- Impersonating staff, developers, or Stylxus

### Daily Mod Checklist

- [ ] Read through `#general` for rule violations
- [ ] Check `#bug-reports` for new reports — react with a checkmark to acknowledge
- [ ] Check `#feedback` for standout suggestions — forward the best to `#dev-notes`
- [ ] Verify `#codes` pinned message is up to date
- [ ] Update `#known-issues` if any bugs have been fixed or new ones confirmed
- [ ] Check `#pet-trading` for scam attempts or suspicious activity

### Responding to Common Situations

**"The game is pay-to-win!"**
> "Game passes give boosts but everything in the game is earnable through gameplay. F2P players can unlock all worlds, hatch all pets, and compete on leaderboards. Passes just speed things up!"

**"When is the next update?"**
> "We're working on updates regularly! Follow #announcements for official news. We don't share exact dates until we're sure, but Update 1 is coming soon."

**"I lost my data!"**
> "Sorry to hear that! Data saves automatically every few minutes. If you lost data, please: 1) Check if you're on the right Roblox account, 2) Try rejoining the game, 3) If it's still missing, report it in #bug-reports with your Roblox username and we'll investigate."

**"Can I be a moderator?"**
> "We select moderators based on community involvement and trust. Stay active, be helpful, and follow the rules — we notice dedicated community members!"

**"Stylxus should add [feature]!"**
> "Great idea! Post it in #feedback so the dev team can see it. Feature requests with the most community support get prioritized."

### Escalation Path

1. **Minor issues** (spam, off-topic, mild rudeness) → Handle directly with warning system
2. **Bug reports** → Acknowledge in `#bug-reports`, log in `#dev-notes` if critical
3. **Data loss reports** → Log in `#dev-notes` immediately, respond to player with investigation timeline
4. **Security issues** (exploits, hacks reported) → Immediately log in `#dev-notes`, do NOT discuss publicly
5. **Server raids/mass spam** → Enable slowmode (30s), lock affected channels, notify all mods in `#mod-chat`
