### Code Distribution Strategy

- **LAUNCH, SPEED** — Display in game description, pinned in Discord
- **STYLXUS90K** — Exclusive to Stylxus YouTube/social content
- **DRIVEFAST** — Twitter/X posts only
- **PETPOWER** — Dropped mid-video in YouTube content
- **REBIRTH1** — Revealed in loading screen tips after 30+ minutes of play
- **NEONHYPE** — TikTok-exclusive, short expiry drives urgency
- **DESERT2X** — Community event reward / Discord giveaway
- **THANKYOU** — Announced when player count milestones are hit
- **UPDATE1** — Announced in update changelog

### Code Design Principles
1. All codes are uppercase, easy to type, memorable
2. Short expiry on social-exclusive codes creates FOMO and urgency
3. Permanent codes (LAUNCH, REBIRTH1) reward organic discovery
4. Milestone codes (THANKYOU) build community loyalty
5. Each code ties to a specific channel — tracks which platforms drive engagement
6. Rewards scale appropriately — social codes are generous but not game-breaking
