## 1. Game Title Options

### Recommended: **Drive a Car Simulator**

The current title is the strongest option. It hits the three highest-traffic keyword clusters on Roblox search: "drive," "car," and "simulator." Changing it would sacrifice existing brand recognition from Stylxus content. Below are alternatives ranked by strategic fit.

| # | Title | Pros | Cons | Verdict |
|---|-------|------|------|---------|
| 1 | **Drive a Car Simulator** | Clear genre signal. All three high-volume keywords ("drive," "car," "simulator") in one title. Matches Stylxus video titles. Reads well at mobile width. | Doesn't mention pets/eggs (secondary hooks). Similar to "Car Driving Simulator" competitors. | **Use this.** Best balance of search volume, readability, and brand consistency. |
| 2 | **Drive a Car Simulator: Pets & Rebirth** | Adds secondary keyword hooks. Differentiates from pure racing sims. Signals depth to browsing players. | 35 characters — long on mobile. Subtitle may get truncated on the Roblox discover feed. Splitting attention across too many concepts. | Strong alternative if truncation isn't an issue. Test on mobile before committing. |
| 3 | **Car Simulator: Drive & Hatch** | Compact. "Hatch" is a high-search term in the Roblox simulator niche (Pet Sim, egg hatching). Two core loops in one title. | Drops "a" which changes the brand cadence. "Car Simulator" alone is generic and has many competitors. | Viable if pivoting brand identity. Not recommended for launch. |
| 4 | **Drive a Car Simulator!** | Exclamation adds energy. Signals fun/casual tone. Otherwise identical to option 1. | Punctuation adds no search value. Some players perceive "!" titles as low-quality. | Minor variant. Use only if A/B testing titles. |
| 5 | **Ultimate Car Simulator** | "Ultimate" is a commonly searched modifier on Roblox. Sounds premium. | Loses the distinctive "Drive a" prefix. Generic — dozens of "Ultimate X Simulator" games exist. No brand connection to Stylxus content. | Not recommended. Too generic. |

### Title Optimization Notes

- **Character count matters.** Roblox truncates titles on mobile at roughly 25-30 characters in the discover feed. "Drive a Car Simulator" at 22 characters displays fully on all devices.
- **Avoid subtitles for launch.** Subtitles get cut on mobile thumbnails and don't index separately for search. If adding keywords, put them in tags and the description instead.
- **Match Stylxus video titles.** Every Stylxus video mentioning the game reinforces the exact title string in YouTube + Roblox cross-search. Changing the title breaks this SEO loop.

## 3. Game Tags

### Recommended Tag Set

Roblox allows selecting a primary genre plus discovery tags/keywords. Use the following configuration:

**Primary Genre:** Simulator

**Sub-genre (if available):** Driving / Vehicle

**Discovery Tags (in priority order):**

| Priority | Tag | Rationale |
|----------|-----|-----------|
| 1 | **Simulator** | Core genre. Highest-traffic category on Roblox. Pet Sim 99, Bee Swarm, and all top competitors use it. |
| 2 | **Driving** | Primary mechanic keyword. Captures players searching specifically for driving games. |
| 3 | **Cars** | Object-specific keyword. High search volume, especially from younger demographics. |
| 4 | **Pets** | Second-highest engagement keyword in the simulator genre. Pet Sim players browsing for similar games. |
| 5 | **Eggs** | Strong simulator-niche keyword. Players who search "eggs" want hatching mechanics — exactly what we offer. |
| 6 | **Tycoon** | Adjacent genre with massive audience overlap. Tycoon players frequently play simulators. Captures browse traffic from the Tycoon sort page. |
| 7 | **Rebirth** | Mechanic-specific keyword. Players who search "rebirth" are experienced simulator players — high-value audience with higher conversion rates. |
| 8 | **Idle** | Captures AFK/passive gameplay seekers. Our Auto-Drive and Auto-Collect passes serve this audience directly. |
| 9 | **Multiplayer** | Signals social features (trading, leaderboards, co-play). Roblox algorithm rewards co-play signals. |
| 10 | **Mobile Friendly** | Roblox's mobile user base is 60%+ of total traffic. Explicitly tagging mobile-friendly captures this browse filter. |

### Tags to Avoid

| Tag | Why Not |
|-----|---------|
| Racing | Implies competitive PvP racing, which we don't have. Players expecting track racing will bounce, hurting play-through rate. |
| Obby | High volume but completely irrelevant. Misleading tags cause fast exits, which the algorithm penalizes. |
| Horror | Irrelevant. Would attract wrong audience. |
| Fighting | No combat mechanic (boss runs are Phase 2). Add this tag only after boss runs launch. |
| Open World | Our highway loop structure isn't open world. Misleading. |

### Keyword Density Strategy

The description already contains these high-value search terms organically:
- "drive" / "driving" — in title + line 1
- "car" / "cars" — in title + features
- "simulator" — in title
- "pets" — in features
- "eggs" / "hatch" — in features
- "rebirth" — in features + line 1
- "coins" — in codes section
- "mobile" — in features

This organic keyword placement reinforces the tags without keyword-stuffing.

## 4. Thumbnail & Icon Creative Briefs

*Visual direction references: `memories/DriveACarSimulator/ConceptArt/visual-style-guide.md`*

### 4.1 Game Icon (512x512)

**Concept: "Speed & Pets"**

| Element | Specification |
|---------|--------------|
| **Background** | Radial gradient: center `#4FC3F7` (sky blue) → outer `#AB47BC` (gem purple). Saturate 15-20% beyond in-game values per style guide §8. |
| **Hero car** | Venom GT or Void Runner in center-left, angled 30° toward viewer. Speed lines trailing behind. Car should fill 40% of the icon area. Apply "Neon City" glow treatment — electric cyan `#29B6F6` rim light on top edge, hot pink `#EC407A` rim on bottom. |
| **Pets** | 2-3 pets orbiting the car: Tabby Cat (bottom-left), Phoenix Chick (top-right), Golden Dragon (right). Each with rarity glow matching their tier. Small enough not to clutter but large enough to read as characters. |
| **Text overlay** | None on the icon itself. Roblox overlays the title below — icon text is redundant and unreadable at 50x50px mobile thumbnail size. |
| **Coin shower** | 5-8 coins (`#FFD54F` gold) scattered as particles around the car. Adds "earning" visual signal. |
| **Border** | No hard border. Subtle vignette (dark edges, 10% opacity) to pop against Roblox's dark UI. |
| **Color boosting** | Per style guide §8: increase saturation 15-20% beyond in-game values. Boost highlights and contrast. The discovery feed compresses images — over-saturate to compensate. |

**Readability test:** The icon must be recognizable at 50x50px (mobile discover feed size). At that scale, players should clearly see: (1) it's a car game, (2) there are pets, (3) it's colorful and fun. Remove any element that doesn't read at thumbnail scale.

## 5. Social Links & Group Setup

### 5.1 Roblox Group

| Field | Recommendation |
|-------|---------------|
| **Group Name** | **Polymita Media** — brand-level group, not game-specific. Allows future games under the same group without renaming. If brand-level already exists, use it. If game-specific preferred: "Drive a Car Simulator Community". |

## Appendix: Discovery Algorithm Alignment

Each deliverable in this document maps to a specific Roblox algorithm signal:

| Algorithm Signal | How This Document Addresses It |
|-----------------|-------------------------------|
| **Qualified play-through rate** (impressions → engaging plays) | Optimized icon (§4.1), compelling description hook (§2), accurate tags (§3) — all reduce misleading impressions and increase qualified clicks. |
| **Playtime per user** | Description front-loads depth (4 worlds, 11 cars, 14 pets) to set expectations for a game with hours of content, not minutes. |
| **Play days per user** | "Weekly updates" promise and active social channels (§5) create return-visit motivation. Code drops tied to specific days drive re-engagement. |
| **Spend days per user** | "13 Game Passes" prominently featured in description. Game pass descriptions in GameCopy.md §4 handle conversion once players are in-game. |
| **Robux spent per user** | Stylxus social proof (§4.5) builds trust, which is a prerequisite for spending. Premium bonus mention (§2) nudges Roblox Premium subscribers to try the game. |
| **Intentional co-play** | "Multiplayer" tag (§3) and Discord community (§5) drive friend-group adoption. Trading system announcement in update roadmap signals social features. |
