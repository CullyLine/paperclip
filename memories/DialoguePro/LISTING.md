## Product Name Options

Five names for the runtime engine (the paid model). The Designer plugin stays "Dialogue Designer."

**Option 1: DialoguePro** *(recommended)*
Direct SEO match for "dialogue roblox", "npc dialogue roblox", "dialogue system roblox". Signals premium quality. Pairs cleanly with the free "Dialogue Designer" plugin. Easy to say, easy to remember.

**Option 2: NarrativeCore**
Professional and scalable — positions this as infrastructure, not a widget. Hits "narrative game roblox" searches. Good if you want room to expand the brand beyond dialogue later (cutscenes, quest systems, etc.).

**Option 3: StoryFlow**
Short, scannable, implies the visual branching concept. Approachable for hobbyist developers and marketable for broader genres beyond RPG. Could pair well with the Designer plugin branding ("StoryFlow + Flow Designer").

**Option 4: Converse**
Clean, single-word, memorable. Less SEO power but strong brand recall. Works well in code: `require(Converse)` reads naturally.

**Option 5: DialogueKit**
Positions it as a toolkit — comprehensive, professional. Broader appeal than "Pro" framing. Aligns well with creator mindset ("I need a kit for dialogue").

## Pricing Recommendation: **1,400 Robux**

**Rationale:**

The V1.0 feature set now competes with paid Unity dialogue plugins that sell for $50–$138 USD:

- Simple text format (dramatically lowers the authoring skill floor)
- 5 built-in themes + `Theme.Extend()` API + per-conversation themes
- Auto-viewport NPC portraits + custom portrait images + per-node model overrides
- Full keyboard, gamepad, and touch input handling
- Auto-advance nodes for cinematic sequences
- Variable interpolation and inline text control
- Free visual Designer plugin as a companion tool

**Justification for 1,400 Robux:**

- Still within the 800–1,500 Robux brief. Pricing at 1,400 (not 1,500) keeps a psychological buffer below the ceiling.
- A higher price in the range signals "professional tool" to the developers most likely to buy it.
- Allows a launch promotion at 999 Robux for early adopters, with room to rise to 1,500 after reviews accumulate.
- Even at 1,400 Robux, dramatically underpriced compared to Unity equivalents — developers feel they got a bargain.

**Alternative:** Launch at **999 Robux** to maximize early install volume and DevForum momentum, with a planned increase to 1,400 after 30 days.
