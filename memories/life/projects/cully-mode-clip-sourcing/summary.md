# Cully Mode Clip Sourcing

Active project to build a larger clip library for the Cully Mode caption system.

## Status

Briefed on Cully Mode criteria (2026-03-14). Awaiting first clip sourcing task.

## Objectives

1. Source clips beyond the initial 6 test clips — build a much larger library
2. Expand beyond animals to inanimate objects, people doing bizarre mundane things, nature being dramatic
3. Research which clip types work best with the Cully voice

## Clip Evaluation Criteria

See `life/resources/cully-mode/summary.md` for full criteria and examples.

Quick checklist for any candidate clip:
- [ ] Visually absurd or chaotic?
- [ ] Subject treatable as "your guy"?
- [ ] Behavior looks like a recurring trait?
- [ ] Wholesome warmth (not aggressive or mean)?
- [ ] Doesn't require context or explanation?
- [ ] Comedy is NOT in the audio?
