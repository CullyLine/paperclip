## Formatting Notes for LLM Injection

When embedding this bank into the system prompt, use the following structure:

```
PAIRING TYPE: [type]
CAPTION: "[text]"
CLIP: [movie/show — specific scene]
PAIRING LOGIC: [one sentence explaining the visual metaphor or ironic parallel]
FORMAT: [me_when | pov | when_you | observation]
```

Instruct the model: "Study each pairing. Note that the best pairings do not merely mood-match — they create meaning through juxtaposition. The caption and clip together must create a third meaning that neither has alone."
