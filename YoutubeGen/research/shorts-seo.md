# YouTube Shorts SEO Research

Research for the YouTube Shorts Generator Pipeline — metadata optimization, algorithm behavior, posting strategy, and thumbnail best practices.

---

## How the Shorts Algorithm Works

### Key Difference from Long-Form

YouTube Shorts uses a **fundamentally different algorithm** from long-form YouTube. The primary difference:

| Signal | Long-Form | Shorts |
|--------|-----------|--------|
| Primary metric | Click-through rate (CTR) | **Swipe-away rate / completion rate** |
| Discovery | Search + recommended | **Shorts shelf + swipe feed** |
| Thumbnail impact | Critical | Minimal (auto-generated in feed) |
| Title visibility | Full title shown | Truncated to ~40 characters |
| Description role | SEO-heavy | Minimal, mainly for hashtags |

### The Two-Phase Distribution Model

1. **Test Phase**: New Shorts are shown to a small audience (100-1,000 views). YouTube measures early signals.
2. **Exploit Phase**: If signals are strong, the Short enters progressively larger distribution waves.

The algorithm re-evaluates at each wave. A Short can go viral days or weeks after posting if it maintains strong signals.

### Critical Metrics (Ranked by Importance)

| Rank | Metric | Target | Why |
|------|--------|--------|-----|
| 1 | **Completion rate** | 70%+ good, 85%+ excellent | The #1 signal. Viewers who watch to the end = strong content |
| 2 | **Swipe-away rate** | Below 30% | How many viewers swipe past in the first 2-3 seconds |
| 3 | **Rewatch rate** | Any rewatches = boost | Signals content is worth seeing again |
| 4 | **Comments** | More = better | Weighted heavier than likes for distribution |
| 5 | **Shares** | More = better | Strongest social signal |
| 6 | **Likes** | More = better | Matters but less than comments/shares |

## Title Optimization

### Rules

- **Front-load the primary keyword** within the first 40 characters (truncation point in feed)
- Keep total length at **40-60 characters max**
- Use **4-6 words** that describe content and spark curiosity
- Numbers, questions, and direct statements outperform vague teasers
- Never clickbait — YouTube penalizes misleading titles

### Effective Title Patterns for This Format

```
"[relatable scenario] 😂" 
"When [specific situation] 💀"
"POV: [scenario]"
"Me vs [thing] #relatable"
"[number] types of [thing]"
"[trending topic] be like..."
```

### Title Examples (for our format)

```
"me explaining my job to my parents 😭"
"POV: your savings after paying rent 💀"
"when the gym motivation lasts exactly one day"
"adulting is just googling how to do stuff"
"my last brain cell during meetings"
```

### Anti-Patterns

- ALL CAPS (looks spammy)
- Excessive emojis (1-2 max)
- Vague titles like "So true" or "Wait for it"
- Misleading titles that don't match content
- Overly long titles that get truncated mid-word

## Hashtag Strategy

### The Formula

Use **3-5 hashtags total**. More than 15 causes YouTube to **ignore all hashtags entirely**.

| Slot | Type | Purpose |
|------|------|---------|
| 1 | `#Shorts` | Required — signals the Shorts shelf |
| 2 | Niche hashtag | Topic-specific: `#relatablehumor`, `#adultingbelike` |
| 3 | Niche hashtag | Format-specific: `#moviereaction`, `#mewhen` |
| 4 | Trending hashtag | Only if genuinely relevant to content |
| 5 | (Optional) Broad | `#funny`, `#viral` — low value but doesn't hurt |

### Placement

- Place hashtags in the **description**, not the title
- The first 3 hashtags appear **above the video title** as clickable links in the YouTube UI
- Order matters: put the most relevant hashtags first

### High-Performing Hashtags for This Format

```
#Shorts #Relatable #MovieReaction #MeWhen #GenZHumor
#AdultingBelike #MovieClip #Sarcasm #POV #DailyStruggle
#TooReal #NoOneTalksAbout #ZoomerHumor #Mood #Vibes
```

## Description Best Practices

### Keep It Short

- **5-15 words** in the description body (not counting hashtags)
- Include 1 primary keyword naturally
- Don't write essay descriptions — nobody reads them on Shorts

### Template

```
[1-2 sentence description with primary keyword] 

#Shorts #[niche1] #[niche2] #[trending if applicable]
```

### Example

```
when the rent check hits different every month 😩

#Shorts #Relatable #AdultingBelike #MovieReaction
```

## Posting Strategy

### Best Times to Post (All Times EST)

| Day | Best Windows | Notes |
|-----|-------------|-------|
| Monday | 11 AM-2 PM, 7-10 PM | Post-weekend engagement dip, evening recovery |
| Tuesday | 11 AM-3 PM, 7-10 PM | Solid midweek engagement |
| Wednesday | 11 AM-3 PM, **4 PM** peak | Wednesday 4 PM is a documented sweet spot |
| Thursday | 11 AM-3 PM, 7-10 PM | Pre-weekend anticipation builds |
| Friday | 12-3 PM, 5-8 PM | Highest engagement day (start of weekend) |
| Saturday | **9-11 AM**, 5-7 PM | Morning catches weekend browsers |
| Sunday | 9-11 AM, 5-7 PM | Similar to Saturday |

### Key Insights

- **12-3 PM EST** is the best universal window for global reach (Europe evening, US afternoon, West Coast lunch)
- **Friday-Sunday** shows highest overall engagement
- Content quality > timing — strong Shorts surface regardless of posting time
- The algorithm tests content over 48+ hours, so suboptimal posting time can be overcome

### Recommended Pipeline Schedule

For a batch of 5 Shorts:

```
Monday    2:00 PM EST — Short 1
Wednesday 4:00 PM EST — Short 2
Friday   12:00 PM EST — Short 3
Saturday  9:00 AM EST — Short 4
Sunday    5:00 PM EST — Short 5
```

## Category Selection

For the movie clip + caption format, these YouTube categories are most appropriate:

| Category | When to Use |
|----------|------------|
| **Comedy** | Default for most captions (relatable humor) |
| **Entertainment** | When the movie clip is the focus |
| **People & Blogs** | For more personal/diary-style captions |

**Recommendation**: Use **Comedy** as the default category. It aligns with the content type and the audience expectation.

## Thumbnail Best Practices for Shorts

### Reality Check

Thumbnails have **minimal impact** on Shorts performance because:
- In the Shorts feed (swipe), thumbnails are never shown — the video auto-plays
- Thumbnails only matter on the channel page and search results
- The algorithm doesn't factor thumbnail CTR for Shorts shelf distribution

### Still Worth Optimizing For

- Channel page browsing (viewers exploring your content)
- YouTube Search results where Shorts appear
- External sharing previews (social media, messaging apps)

### Guidelines

| Rule | Detail |
|------|--------|
| Close-up subjects | 3x higher CTR than wide shots |
| High contrast | Bold colors, clear focal point |
| Text overlay | Short (2-4 words max), high contrast |
| Expression | Expressive face from the movie clip = eye-catching |
| Consistency | Same style across Shorts = brand recognition |

### Thumbnail Generation Strategy

For the pipeline:

1. **Extract the most expressive frame** from the movie clip (peak emotion moment)
2. **Overlay the caption text** — shortened to 3-5 key words
3. Use **white text with black outline/shadow** for readability
4. Crop to **16:9** aspect ratio (YouTube's thumbnail format, not the video's 9:16)
5. Target file size: under 2MB, 1280×720 minimum

## Algorithm Gaming — What NOT to Do

These tactics are penalized or ineffective:

| Tactic | Why It Fails |
|--------|-------------|
| Sub4sub / engagement pods | YouTube detects artificial engagement patterns |
| Spam posting (10+ Shorts/day) | Cannibalizes your own reach, algorithm fatigue |
| Misleading titles/thumbnails | Increases swipe-away rate, kills distribution |
| Deleting and re-uploading | Resets metrics, no benefit |
| Using copyrighted music | Content ID strikes, potential channel termination |
| Hashtag stuffing (15+) | YouTube ignores ALL hashtags |

## Metadata JSON Template

For the Metadata Generator module output:

```typescript
interface ShortsMetadata {
  title: string;          // 40-60 chars, front-loaded keyword
  description: string;    // 5-15 words + hashtags
  hashtags: string[];     // 3-5 hashtags, #Shorts always first
  category: "Comedy" | "Entertainment" | "People & Blogs";
  scheduledTime?: string; // ISO timestamp, optimal posting window
  thumbnail: {
    sourceFrame: number;  // timestamp in seconds for frame extraction
    overlayText: string;  // shortened caption (3-5 words)
  };
  tags: string[];         // YouTube tags (less important but doesn't hurt)
}
```
