# Metadata Generation Templates
## YouTube Shorts Generator — Titles, Descriptions, Hashtags

---

## Overview

YouTube Shorts metadata has two jobs: **algorithm discovery** (search + suggested) and **human click-through** (the title makes someone tap). These are sometimes in tension — this guide resolves that.

Key Shorts-specific facts:
- Titles display truncated at ~60 characters in browse; full title shown on video page
- Descriptions are largely invisible in Shorts UI but indexed for search
- First 3 hashtags in description appear as tappable overlays on the short
- Tags (keywords) are less important than hashtags + title for Shorts discovery
- Watch time % matters more than absolute time — a 12-second short watched twice is better than a 45-second short abandoned at 15s

## Title Templates

### Template A: "Me When" Direct
```
me when [situation] 😭
```
> `me when the minimum payment IS the interest 😭`
> `me when sunday lasts 11 minutes 😭`
> `me when my savings account does its thing 😭`

**Use when:** Caption is already a "me when" — title mirrors it for algorithm consistency

---

### Template B: POV Scene Title
```
pov: [situation] 💀
```
> `pov: you opened linkedin by accident 💀`
> `pov: it's sunday night again 💀`
> `pov: explaining your job to your parents 💀`

**Use when:** Caption uses POV format

---

### Template C: The Observation + Hashtag Anchor
```
[dry observation] #[trending topic tag]
```
> `the audacity of a bill on top of another bill #adulting`
> `figuring it out has no end date apparently #mentalhealth`
> `mandatory fun is mandatory #workmemes`

**Use when:** Caption is a standalone observation tied to a trending category

---

### Template D: Relatable Spiral (higher click-through, lower SEO)
```
this is [too/extremely/painfully] relatable [emoji]
```
> `this is extremely relatable if you have rent 😅`
> `this is too real and I didn't ask for it 💀`

**Use when:** High-confidence caption, prioritizing human click-through over search

---

### Template E: Question Hook
```
who else [situation]? [emoji]
```
> `who else calculated if they can afford to get sick? 😭`
> `who else has been "almost done" for 4 hours? 💀`

**Use when:** Caption is particularly universal — the question amplifies community feeling

---

### Title Rules
- **Length:** 50–70 characters for full display; aim for key info in first 50
- **Casing:** all lowercase preferred (matches zoomer voice); avoid Title Case
- **Emoji:** 1–2 max, always at end — `😭` `💀` `😅` `💀` work best for this content
- **No clickbait promises** ("You won't believe...", "Watch till the end")
- **No hashtags in title** — use description for hashtags
- **Include topic keyword** when possible for search indexing

## Description Templates

### Template 1: Minimal (Recommended for Most Shorts)
```
[caption text, verbatim]

#[primary topic] #[mood/emotion category] #[format tag] #Shorts #[secondary topic]
```

**Example:**
```
me when the minimum payment IS the interest

#adulting #financialstress #relatable #Shorts #personalfinance
```

---

### Template 2: Engagement Prompt
```
[caption text]

can you relate? 👇

#[tag1] #[tag2] #[tag3] #Shorts #[tag4]
```

**Example:**
```
pov: you opened linkedin by accident

can you relate? 👇

#workmemes #linkedin #officehumor #Shorts #careertok
```

---

### Template 3: Film Reference (when scene is prominent/recognizable)
```
[caption text]

scene: [movie/show name] ([year])

#[movie tag] #[mood tag] #[topic tag] #Shorts #filmtok
```

**Example:**
```
when you've been "almost done" for 4 hours

scene: The Office (US)

#theoffice #workmemes #relatable #Shorts #officetok
```

---

### Template 4: Trending Topic Anchor
```
[caption text]

[1-line context about why this is trending right now]

#[trending hashtag] #[emotion tag] #[format tag] #Shorts #[secondary]
```

**Example:**
```
me calculating if I can afford to be sick this month

healthcare costs trending after [recent news context]

#healthcarecosts #adulting #mentalhealth #Shorts #zoomers
```

## Hashtag Strategy

### Core Hashtag Tiers

**Tier 1 — Always Include (1–2 per video)**
High-volume, broad discovery. Low competition resistance but essential baseline:
- `#Shorts`
- `#relatable`

**Tier 2 — Category Tags (2–3 per video)**
Pick based on the caption topic:

| Topic Area | Recommended Tags |
|---|---|
| Money / Finance | `#adulting` `#personalfinance` `#financialstress` `#broke` |
| Work / Career | `#workmemes` `#officememes` `#9to5` `#careertok` |
| Existential / General | `#mentalhealth` `#zoomers` `#genz` `#quarterlifecrisis` |
| Relationships | `#dating` `#relationships` `#situationship` |
| Current Events | Use the specific trending tag + `#news` |

**Tier 3 — Movie/Show Reference (1–2 when applicable)**
Only add if the film is prominent and recognizable:
- `#theoffice` `#breakingbad` `#succession` `#filmtok` `#movieclips`

**Tier 4 — Emerging/Niche (1 per video, rotate)**
Smaller communities, higher engagement rate:
- `#quarterlifecrisis` `#adulting101` `#zillennials` `#realtalk`

### Hashtag Placement Rules
- First 3 hashtags appear as tappable overlays on Shorts — make them count
- Recommended order: `#[specific topic]` `#relatable` `#Shorts` (then others)
- Total hashtags: 5–8 (more than 8 looks spammy and may suppress reach)
- Never use hashtags unrelated to the content — algorithm penalizes hashtag stuffing

---

## Tags (Keywords) Field

Less critical for Shorts, but still worth populating:

```
[topic keywords], relatable, funny, zoomer humor, adulting, gen z, [movie/show name if featured], short video, viral, comedy, [specific trending topic keywords]
```

**Example (finance topic):**
```
adulting, finances, rent, student loans, funny, relatable, gen z humor, broke millennial, personal finance comedy, short video, viral short
```

**Example (work topic):**
```
work memes, office humor, the office, relatable work, 9 to 5, career, funny work, gen z at work, monday, corporate humor
```

---

## Category Recommendations

| Content Type | YouTube Category |
|---|---|
| All standard Shorts (this format) | **Comedy** (primary) |
| Shorts featuring film clips | **Film & Animation** (if film focus is strong) |
| Trending current events content | **News & Politics** (only if topic is directly political) |

**Default:** Use **Comedy** for >90% of content. The algorithm already classifies Shorts based on engagement patterns — the category is a weak signal.

## Metadata Generation — LLM Output Schema

When calling the Metadata Generator module, expect this JSON structure:

```json
{
  "title": "string (50–70 chars)",
  "description": "string (multi-line, includes caption + hashtags)",
  "hashtags": ["string", "string", "string", "string", "string"],
  "tags": ["string", "string"],
  "category": "Comedy",
  "titleAlternatives": [
    "string (alt title option 1)",
    "string (alt title option 2)"
  ]
}
```
